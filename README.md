# AdCue Demo

AdCue is a Tencent Video Ads Challenge demo: ads that know when to appear.

## Demo Pages

- `index.html`: full product logic demo
- `recording.html`: hands-on comparison demo for judges and screen recording
- `AdCue_Product_Design_Document.pdf`: product design document

## Local Preview

```bash
node local-server.mjs
```

Then open:

```text
http://127.0.0.1:4174/recording.html
```

The demo uses `01-demo.mp4`, a compressed node-based clip for web preview.
